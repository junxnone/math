# 图论 (Graph theory)

  - 图 (Graph) 是由若干给定的顶点及连接两顶点的边所构成的图形，这种图形通常用来描述某些事物之间的某种特定关系。
  - \*\*顶点(Vertex/Node)**用于代表事物，连接两顶点的**边(Edge)\*\*则用于表示两个事物间具有这种关系。
  - 有限图/无限图
  - 有向图/无向图/混合图
  - 有向边/无向边
  - 阶(order)
  - 相邻
