# 向量

## 向量点乘

  - 点乘 - Dot Product/Scalar Product
  - $a \\cdot b = \\sum\_{i=1}^{n} a\_{i}b\_{i}$ - $a = \[a\_{1},
    a\_{2}, ..., a\_{n}\]$ - $b = \[b\_{1}, b\_{2}, ..., b\_{n}\]$
  - $a \\cdot b = |a||b|cos\\theta $
  - $a \\cdot b = b \\cdot a $
  - $a \\cdot (b + c) = a \\cdot b + a \\cdot c$

## 向量叉乘

  - 叉乘 - Cross Product/Vector Product
